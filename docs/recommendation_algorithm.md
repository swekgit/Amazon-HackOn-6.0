# Recommendation Algorithm

Input:

User Tags

Example:

[
 "nike_affinity",
 "premium_buyer",
 "clothing_lover"
]

Product Tags

Example:

[
 "nike",
 "premium",
 "clothing",
 "shirt"
]

Scoring:

Brand Match = 30

Category Match = 30

Price Match = 20

Purchase History Match = 20

Total = 100

Products are sorted by score.

Highest score appears first.
